## Recommendations

### Current Usage (index.md):
- Using **original** generated charts (`model-comparison.png`, `per-class-accuracy.png`)
- Using **original** confusion matrix (`resnet50.png`)
- Using **user-provided** sample images in hero grid

### Optional Enhancements:
- Switch to **themed** charts for more polished presentation (`*-themed.png`)
- Add `landing-hero-poster.png` as hero section background
- Integrate section visuals (`pipeline-themed.png`, `demo-interface-themed.png`)

### Size Optimization Needed:
- Sample images are quite large (1.8-2.5MB each)
- Consider compressing to ~500KB each for web performance
- Use tools like TinyPNG or ImageOptim
